# Four Stage Grid Layout Guide

Use this when the output is a 2 by 2 evolution atlas.

- The canvas contains four equal cells in reading order.
- Each cell shows the same asset identity at a different upgrade stage.
- Stage changes should be clear through silhouette, scale, ornaments, power effects, or material complexity.
- Keep the camera angle, lighting, palette, and line weight identical across all cells.
- Keep seams clean so each cell can be sliced into a separate sprite.

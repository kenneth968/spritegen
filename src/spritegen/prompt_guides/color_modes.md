# Color Mode Prompt Guide

Use color treatment as a production rule, not decoration.

- Full color uses the project palette naturally while preserving readable values.
- Limited palette uses only the listed project colors and their necessary highlights or shadows.
- Black and white uses clear inked silhouettes, white fills, and no hue information.
- Grayscale value-map treats shades as coded material or lighting indices; keep values stepped and separable.
- Single-hue value-map uses one base hue with controlled tint and shade steps, useful for recoloring or shader workflows.
- Never ignore the color mode when it conflicts with generic style words.
